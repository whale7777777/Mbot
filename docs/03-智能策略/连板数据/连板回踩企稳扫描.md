## 二、企稳信号含义

| 信号 | 计算逻辑 | 盘面含义 |
|------|----------|----------|
| 跌幅收窄 | 最新绿K跌幅，比前一绿K绝对值小 ≥0.3pp | 杀跌力度减弱，恐慌盘减少。 |
| 缩量 | 最新绿K成交量 < 前一绿K × 90% | 抛压减轻，筹码趋于稳定。 |
| 低点抬高 | 最新绿K最低价 ≥ 前一绿K最低价 | 不再创新低，有横向支撑。 |
| 小阴线 | 最新绿K涨跌幅绝对值 ≤ 2.5% | 小幅整理，多空暂时平衡。 |
| 下影偏长/收偏上 | 收盘位于当日振幅的上 45% 区域 | 盘中下探后有承接，收在相对高位。 |
| 未破调整前低 | 最新绿K最低价 ≥ 绿K段前一日最低价 | 调整未加深，结构未被破坏。 |
