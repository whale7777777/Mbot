## 三、值得关注 / 可交易标的

> 冰点反抽原则：**先辨认有宽度的主线，再谈仓位；不追一字高标。** 以下为研究观察，不构成投资建议。

### 3.1 连板池内（相对优先）

**立新能源（001258，3 板）** — 电力链空间龙 / 总龙头观察  
 └ 同线：华银电力(2板，中军)、乐山电力(2板，补涨)；首板：涪陵电力、华电能源、金开新能、晋控电力  
 └ 似近月龙头：大有能源  
 └ 理由：上周五唯一有宽度的方向今日继续晋级；自身成交与封单在 3 板中最好。注意炸 2、弱转强——只适合**回封/低开确认**，不宜盲目高开排板。

**共进股份（603118，2 板）** — AI/算力簇可交易中军  
 └ 同线：九安医疗(2板，空间/叙事中军)、米奥会展(2板，同线弱标慎追)  
 └ 似近月龙头：天娱数科  
 └ 理由：数据中心交换机+机器人叙事，秒板强封、换手适中；比九安更容易成交。若 AI 首板继续扩散可留，否则当作反抽轮动。

**华银电力（600744，2 板）** — 电力中军 / 流动性补涨  
 └ 同线：立新能源(3)、乐山电力(2)+电力首板群  
 └ 似近月龙头：大有能源  
 └ 理由：二板里成交额最大；央企火电+新能源，适合作为立新的替补观察，不抢空间龙溢价。

**慎追 / 难买**

- 艾艾精工（5 板）：空间龙外观，实为控股权一字，**非题材总龙头**；封死难成交。  
- 九安医疗（2 板）：AI 叙事强但一字厚封，**买不进**则换成共进。  
- 米奥会展（2 板）：同线但炸 3、弱转强，**质量弱，非首选**。  
- 乐山电力（2 板）：同线跟风、封单薄，优先立新/华银。  
- 金牛化工：缺跟风群，优先级靠后。
