# 连板模拟盘

> 初始资金 **20,000** 元 | 策略：晋级概率 + 封板质量 + 主线簇 + **量能/风格风控**

## 机器人逻辑

每日收盘后读取 `每日/YYYYMMDD.json` 连板池，自动执行：

1. **卖出**：持仓股未晋级（T 日 n 板 → T+1 未达 n+1 板）则次日卖出
2. **买入**：对候选股评分，选评分 ≥ 55 且满足风控的标的
3. **记录**：写入本目录操作文档

### 评分因子（与连板策略一致）

| 加分 | 减分 |
|------|------|
| 晋级概率高 | 炸板 ≥ 3 次 |
| 强封 / 零炸板 | 一字板（换手 < 0.5%） |
| 封单占比 ≥ 20% | 晋级率 < 28% |
| 主线簇 ≥ 2 只 | 连板数 > 4 |

### 风控（v2：量能 + 风格 + 情绪）

| 规则 | 默认阈值 | 行为 |
|------|----------|------|
| **量能参考** | 上证成交额 < **9000 亿** | 仅标注「窒息/冰点」，**不禁止开仓**（可配合缩量降仓） |
| **缩量观望** | 上证成交额 < **10000 亿** | 单票仓位 ×0.75 |
| **指数退潮** | 5 日跌超 **3%** 且量能萎缩 | 强制冰点情绪，仓位上限 10% |
| **主线簇** | 同题材连板池 ≥ **3 只** | 否则不买（过滤独苗高标） |
| **冰点高位** | 连板池 ≤ 6 只 | 不买 **≥3 板** |
| 最多持仓 | 2 只 | |
| 单票上限 | 总资产 40%（窒息期 10%） | |
| 冰点总仓 | 连板池 ≤ 6 只时 ≤ 50% | |
| 偏暖总仓 | 连板池 > 6 只时 ≤ 80% | |

配置项见 `paper_config.json`。

## 文件说明

| 文件 | 说明 |
|------|------|
| `paper_config.json` | 参数配置（资金、仓位、评分阈值、量能风控） |
| `持仓.json` | 当前现金与持仓 |
| `成交记录.json` | 全量买卖流水 |
| `操作记录.md` | 汇总日志 |
| `每日/YYYYMMDD.md` | 当日盘面观察与操作 |

## 命令

```bash
# 每日运行（先确保当日连板数据已生成）
python scripts/lianban.py today
python scripts/lianban.py paper

# 或
python scripts/lianban_paper.py run
python scripts/lianban_paper.py run --date 20260716

# 历史回测模拟
python scripts/lianban_paper.py backfill --from 20260710 --to 20260720 --reset

# 查看状态 / 重置
python scripts/lianban_paper.py status
python scripts/lianban_paper.py reset
```
